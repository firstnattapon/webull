ส่วนเกินทุนจาก Rebalancing

ทดลองสุ่มราคา 100 รอบ เปรียบเทียบกระแสเงินสดจาก Rebalancing กับเส้นอ้างอิง ln และนำส่วนเกินสะสมไปวางบนเส้นอ้างอิงตามระดับราคาอย่างถูกความหมาย

1) เส้นอ้างอิงทางทฤษฎี

Rₙ = Fix_c × ln(Pₙ / P₀)

เป็นกระแสเงินสดอ้างอิงของการรักษามูลค่าสินทรัพย์คงที่แบบต่อเนื่อง: ค่าบวกคือรับเงินจากการขาย และค่าลบคือใช้เงินซื้อ

2) เส้น Rebalancing จริง

Aₙ = Fix_c × Σ [Pᵢ / Pᵢ₋₁ − 1] Eₙ = Aₙ − Rₙ

Aₙ สะสมผลจากทุกช่วงราคาที่เกิดขึ้นจริง ส่วน Eₙ คือเงินเกินทุนสะสมเหนือเส้นอ้างอิง

กราฟที่ 2 — เงินทุนเทียบกับระดับราคา (แก้ไข)

แกน X: ราคา x ตั้งแต่ 0 ถึง 2t₀ Y₁(x) = Fix_c × ln(x / t₀) ← เส้นอ้างอิง Y₂(x) = Y₁(x) + Eₙ ← เส้น Rebalancing + เงินเกินทุนสะสม
Y₁: เส้นอ้างอิง lnY₂: อ้างอิง + Eₙ

จุดสำคัญ: แกนราคาเริ่มแสดงที่ 0 ตามโจทย์ แต่ไม่ลากเส้นที่ x = 0 เพราะ ln(0) ไม่มีค่าจำกัด (มุ่งสู่ −∞) การคำนวณเส้นจึงเริ่มที่ค่าบวกเล็ก ๆ ใกล้ศูนย์ ส่วน Y₂ เป็นเส้น Y₁ ที่เลื่อนขึ้นในแนวตั้งเท่ากับ Eₙ จึงมีช่องว่างคงที่เท่ากับส่วนเกินสะสม

การอ่านเครื่องหมาย: ตามสมการ ln(x/t₀) ค่าบวกหมายถึงเงินสดที่ได้รับจากการขาย และค่าลบหมายถึงเงินสดที่ใช้ซื้อ หากต้องการให้ “เงินทุนที่ใช้ซื้อ” เป็นบวก ต้องกลับเครื่องหมายเป็น Fix_c × ln(t₀/x) ซึ่งเป็นคนละ convention
